# 📊 Flipkart Mobile Dataset Analysis (Power BI)

## 📌 Project Overview
This project analyzes Flipkart mobile phone data to understand brand performance, customer ratings, reviews, discounts, and sales insights.  
The analysis is performed using **Power BI** to create an interactive dashboard that helps identify trends and compare different mobile brands.

The dashboard provides a clear view of the **top brands, most reviewed mobiles, highest-rated products, and discount trends**.

---

## 🎯 Objectives
- Analyze mobile brand performance on Flipkart
- Identify top brands based on **ratings, reviews, and discounts**
- Understand customer feedback through ratings and reviews
- Visualize sales insights using interactive dashboards
- Support **data-driven decision making**

---

## 🛠️ Tools & Technologies Used
- **Power BI** – Dashboard and Data Visualization  
- **Microsoft Excel** – Data Source  
- **Data Cleaning** – Power Query  
- **Data Analysis** – DAX & Aggregations  

---

---

## 📈 Dashboard Features
The Power BI dashboard includes:

- Total Brands and Total Mobiles
- Total Sales Analysis
- Average Discount Percentage
- Average Rating of Mobiles
- Top 5 Brands by Ratings
- Top 5 Brands by Reviews
- Top 5 Brands by Discount
- Top 5 Mobiles by Ratings
- Top 5 Mobiles by Reviews
- Interactive Filters (Brand & Mobile)

## 🔍 Key Insights
- **Samsung** has the highest share of customer reviews.
- **Redmi** shows strong performance in ratings.
- **Xiaomi models** dominate in review counts.
- Certain mobiles offer **high discounts but moderate ratings**, indicating price-driven purchases.
- Brands with higher ratings generally receive more customer engagement.

## 🚀 How to Use
